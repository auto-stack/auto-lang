# UI Gallery

24 shadcn-vue building block replicas — login forms, dashboards, product tables, and sidebar layouts.

### <a id="charts"></a><a href="/ui/charts/index.html" target="_self">Charts ↗</a>

6 shadcn-vue chart replicas — area charts, bar charts, line charts, and donut charts powered by Unovis.
