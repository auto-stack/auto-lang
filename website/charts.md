# Charts

Explore shadcn-vue chart components built with Auto — area charts, bar charts, line charts, and donut charts powered by Unovis.

<a href="/ui/charts/index.html" target="_self">Open Charts Gallery ↗</a>

---

## Web

Powered by Vue 3 + shadcn-vue + Unovis, auto-generated via `auto build --backend vue`.

### Supported Chart Types

| Auto Tag | Component | Description |
|----------|-----------|-------------|
| `area-chart` | AreaChart | Filled area charts with gradient support |
| `bar-chart` | BarChart | Grouped and stacked bar charts |
| `line-chart` | LineChart | Trend lines with curve type options |
| `donut-chart` | DonutChart | Donut and pie charts |
