# UI

Auto provides a comprehensive UI framework that generates native components for multiple platforms from a single source.

## Web

- [**Component Gallery**](/ui/gallery/index.html) — 46+ shadcn-vue components
- [**Blocks Gallery**](/ui/blocks/index.html) — Pre-built page blocks (login, dashboard, sidebar, etc.)
- [**Charts Gallery**](/ui/charts/index.html) — Area, bar, line, donut charts with Recharts

## Native Platforms

- [**Desktop**](/ui-desktop) — Cross-platform desktop apps (Windows, macOS, Linux)
- [**Android**](/ui-android) — Native Android applications
- [**Harmony**](/ui-harmony) — HarmonyOS applications

All UI components are generated from Auto source code using the `auto ui` CLI command.
