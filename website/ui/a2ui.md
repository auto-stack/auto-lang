# A2UI

A2UI is the Auto UI framework — a comprehensive system for generating native user interfaces across multiple platforms from a single Auto source.

## Overview

A2UI compiles Auto `view` declarations to:

- **Web** — Vue 3 + shadcn-vue (via `auto build --backend vue`)
- **Desktop** — Tauri / Winit / LVGL (planned)
- **Android** — Jetpack Compose (planned)
- **HarmonyOS** — ArkTS (planned)
