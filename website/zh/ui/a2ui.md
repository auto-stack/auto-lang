# A2UI

A2UI 是 Auto UI 框架 —— 一个全面的系统，可以从单一的 Auto 源码生成多平台的原生用户界面。

## 概览

A2UI 将 Auto `view` 声明编译到：

- **Web** — Vue 3 + shadcn-vue（通过 `auto build --backend vue`）
- **桌面端** — Tauri / Winit / LVGL（计划中）
- **Android** — Jetpack Compose（计划中）
- **鸿蒙** — ArkTS（计划中）
