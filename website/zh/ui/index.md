# UI

Auto 提供了一套全面的 UI 框架，可以从单一源码生成多平台的原生组件。

## Web

- [**组件画廊**](/ui/gallery/index.html) — 46+ 个 shadcn-vue 组件
- [**区块画廊**](/ui/blocks/index.html) — 预构建的页面区块（登录、仪表盘、侧边栏等）
- [**图表画廊**](/ui/charts/index.html) — 面积图、柱状图、折线图、环形图，基于 Recharts

## 原生平台

- [**桌面端**](/zh/ui-desktop) — 跨平台桌面应用（Windows、macOS、Linux）
- [**Android**](/zh/ui-android) — 原生 Android 应用
- [**鸿蒙**](/zh/ui-harmony) — 鸿蒙应用

所有 UI 组件都通过 `auto ui` CLI 命令从 Auto 源码生成。
