# 图表

探索使用 Auto 构建的 shadcn-vue 图表组件 —— 面积图、柱状图、折线图和环形图，由 Unovis 提供支持。

<a href="/ui/charts/index.html" target="_self">打开图表画廊 ↗</a>

---

## Web

基于 Vue 3 + shadcn-vue + Unovis，由 `auto build --backend vue` 自动生成。

### 支持的图表类型

| Auto 标签 | 组件 | 说明 |
|----------|------|------|
| `area-chart` | AreaChart | 支持渐变的面积图 |
| `bar-chart` | BarChart | 分组和堆叠柱状图 |
| `line-chart` | LineChart | 支持曲线类型的折线图 |
| `donut-chart` | DonutChart | 环形图和饼图 |
