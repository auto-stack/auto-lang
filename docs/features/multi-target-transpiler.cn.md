# 多目标转译器

Auto 被设计为多目标转译器，可编译为多种主流语言，实现真正的"一次编写，到处运行"。

## 支持的目标

- **C** — 用于嵌入式系统和裸机编程
- **Rust** — 用于具有内存安全的系统编程
- **TypeScript** — 用于 Web 和 Node.js 应用
- **Python** — 用于数据科学和快速原型开发

## 零成本抽象

Auto 的转译器在保留高级抽象语义的同时，不产生运行时开销。泛型、枚举和模式匹配等概念都会被编译为高效的目标语言代码。

## 使用场景

- 需要在多个生态系统中工作的跨平台库
- 需要 C 级别速度的性能关键代码
- 编译为 TypeScript 的 Web 应用程序
- 在 Python 中快速原型开发，然后编译为 C 用于生产环境
