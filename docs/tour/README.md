# Auto Language Tour

Welcome to the **Auto Language Tour** — a hands-on journey through every major feature of Auto.

Each chapter includes runnable `.at` code examples. Use `auto run <file>.at` to try them yourself.

## Chapters

1. [Hello, Auto!](./ch01-hello) — Printing, variables, f-strings
2. [Types](./ch02-types) — Primitives, structs, enums, tags
3. [Functions](./ch03-functions) — Parameters, returns, closures
4. [Control Flow](./ch04-control) — if/else, loops, break
5. [Pattern Matching](./ch05-patterns) — `is` expressions
6. [Error Handling](./ch06-errors) — `!` functions, `.?` propagation
7. [Collections](./ch07-collections) — Arrays, lists, maps
8. [Methods & Specs](./ch08-methods) — ext, spec, traits
9. [Generics](./ch09-generics) — Type parameters, specs
10. [Modules](./ch10-modules) — use, imports, annotations
11. [Async](./ch11-async) — Futures, await, tasks
12. [Interop](./ch12-interop) — FFI, C/Rust/Python interop
