```
└─────────────────────────────────────────────────────────┘
```

## Implementation Tasks

### Task 1: Generate Rust Structs with Serde

**File:** `crates/auto-lang/src/trans/rust.rs`

Generate from `back/api.at` type definitions:

```rust
// Generated from: pub type User = { id: int, name: str, email: str }
#[derive(Clone, Debug, serde::Serialize, serde::Deserialize)]
pub struct User {
    pub id: i64,
    pub name: String,
    pub email: String,
}
```

**Key decisions:**
- `int` → `i64`, `str` → `String`
- Add `serde::Serialize, serde::Deserialize` derives for JSON
- Use `#[repr(Rust)]` (default) - no ABI concerns for a2rs

### Task 2: Generate Axum Routes from `#[api]` Functions

**File:** `crates/auto-man/src/api_gen.rs` (new or enhance existing)

Generate from `back/api.at`:

```rust
// Generated from: #[api(method = "GET", path = "/api/users")]
pub async fn listusers() -> axum::Json<Vec<User>> {
    axum::Json(crate::db::all_users())
}

// In main():
let app = axum::Router::new()
    .route("/api/users", axum::routing::get(listusers));
```

**Key decisions:**
- Parse `#[api(method, path)]` annotation
- Generate axum route handler
- Register in generated `main.rs`

### Task 3: Generate TypeScript API Client

**File:** `crates/auto-man/src/api_gen/ts_client.rs` (new)

Generate to `dist/src/lib/api.ts`:

```typescript
// Generated from back/api.at
export interface User {
  id: number
  name: string
  email: string
}

export async function listusers(): Promise<User[]> {
  const response = await fetch('http://localhost:3000/api/users')
  if (!response.ok) throw new Error(`HTTP ${response.status}`)
  return response.json()
}
```

**Key decisions:**
- `int` → `number`, `str` → `string`
- Use native `fetch()` (browser-compatible)
- Hardcode `localhost:3000` for demo (configurable later)

### Task 4: Update `front/app.at` to Display Users

**File:** `examples/api-example/front/app.at`

```auto
widget App {
    msg Msg { Click, Load }

    model {
        var count int = 0
        var users []User = []
    }

    view {
        col {
            h1 (text: "API Demo") {}

            row {
                text `Count: ${.count}`
            }

            button (text: "Load Users", onclick: .Load) {}

            // Display users
            for user in .users {
                row {
                    text `${user.name} (${user.email})`
                }
            }
        }
    }

    on {
        .Click => {
            .count = .count + 1
        }
        .Load => {
            .users = Api.listusers()
        }
    }
}
```

**Key decisions:**
- Add `users []User` to model
- Call `Api.listusers()` on Load
- Display users in a loop

### Task 6: Generate Rust Server Entry Point

**File:** `crates/auto-man/src/build.rs` (enhance)

Generate `rust/src/main.rs`:

```rust
mod api;
mod db;

#[tokio::main]
async fn main() {
    let app = axum::Router::new()
        .route("/api/users", axum::routing::get(api::listusers));

    let listener = tokio::net::TcpListener::bind("127.0.0.1:3000").await.unwrap();
    axum::serve(listener, app).await.unwrap();
}
```
